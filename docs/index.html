g2

Using g2 with μJam is easy. A custom html element <g-2> with a predefined set of supported attributes can be injected in html as well as in md documents.

On top of that we get seamless integration into the markdown preview window of VSCode.

You can see this here live. We can also combine <g-2> elements with inline scripts.
